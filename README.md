我的Emacs配置文件和插件
=======================

1、Emacs配置文件
----------------
配置文件为.emacs文件

2、Emacs插件
-------------
插件安装目录为~/.emacs.d/plugins/

插件如下：
 - [AUCTeX](http://www.gnu.org/software/auctex/)
   Version: 11.86
 - [Auto-Complete](http://cx4a.org/software/auto-complete/)
   Version: 1.3.1
 - [Org-Mode](http://orgmode.org/)
   Version: 7.8.11
 - [YASnippet](https://github.com/capitaomorte/yasnippet)
   Version: 0.7.0
