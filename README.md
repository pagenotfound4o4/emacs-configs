我的Emacs配置文件和插件
=======================
Author: Unlucky<br/>

1、目录结构
----------------
 - .emacs 主配置文件
 - /.emacs.d/plugins 存放插件代码
 - /.emacs.d/settings 存放插件配置文件
 - /bin 存放Windows下Emacs显示图片所需DLL

2、Emacs插件
-------------
插件安装目录为~/.emacs.d/plugins/
